## Features

This extension provides support for Synulb.
It provides syntax highlighting and allows Synulb to run in VS Code

In order for this extension to work, you have to run Installer.py
Then, to run your program, you have to use Ctrl+Shift+P and select "Run Synulb".

## Extension Settings

Include if your extension adds any VS Code settings through the `contributes.configuration` extension point.

For example:

This extension contributes the following settings:

* `myExtension.enable`: Enable/disable this extension.
* `myExtension.thing`: Set to `blah` to do something.
